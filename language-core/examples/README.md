# Examples

Contoh penggunaan library language-core.

- `app-main.fm`: contoh entry UI `.fm`
- `component-header.fm`: contoh komponen reusable
- `app-controller.fl`: contoh logic controller lintas web/desktop
- `session-service.fl`: contoh logic service
- `web-adapter.fl`: contoh adapter web (Browser runtime)
- `desktop-adapter.fl`: contoh adapter desktop (Desktop runtime)
