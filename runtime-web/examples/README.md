# Examples

Contoh penggunaan library runtime-web.

Catatan:

- Verifikasi `app.css` untuk memastikan style sudah keluar dalam bentuk canonical.
- Jika project punya FL logic bridge, verifikasi `runtime/app/50_actions_state.js` sudah memuat `formoGeneratedActions`.
- Jika build web memakai `--strict-parity`, cek kemungkinan file `desktop.parity.json` sebagai hasil audit parity desktop.
