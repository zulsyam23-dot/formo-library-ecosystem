# Features: knowledge-pack

Daftar fitur baseline knowledge pack yang wajib tersedia di library ini.

## Fitur Wajib

1. `domain_glossary`
   - Istilah domain Formo agar AI tidak salah interpretasi.
2. `prompt_patterns`
   - Pola prompt dan output target untuk use case umum.
3. `integration_checklists`
   - Checklist singkat untuk integrasi editor, CI, dan pipeline AI agent.

## Catatan

- Library ini masih `bootstrap`.
- Fokus saat ini dokumentasi dan pattern reuse.
