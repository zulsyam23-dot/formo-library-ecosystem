# Features: ai-interop

Daftar fitur baseline AI interop yang wajib tersedia di library ini.

## Fitur Wajib

1. `capability_profile`
   - Profil kemampuan library dalam format machine-readable.
2. `normalized_error_envelope`
   - Format error generik yang stabil untuk AI/tool.
3. `machine_readable_registry`
   - Registry ekosistem yang dapat diindeks tanpa parsing dokumen bebas.

## Catatan

- Library ini masih `bootstrap`.
- Fokus saat ini adalah kontrak data, bukan runtime model inference.
